Azure DocumentDB

Microsoft Azure QuickStarts

The sample demonstrates how to leverage DocumentDB to store JSON documents.

If you don't have a Microsoft Azure subscription you can get a FREE trial account here.

This sample demonstrates the following scenarios:

  1. How to connect to a DocumentDB Account
  2. How to query for a Database by its Id, and create one if not found
  3. How to query for a DocumentCollection by its Id, and create one if not found
  4. How to create a Document
  5. How to query for Documents
  6. How to delete a Database and a DocumentCollection

Running this sample

  1. Use the Azure Portal to create a new DocumentDB account.
  2. Edit app.config and replace the serviceEndpoint and authKey with values provided from the portal
  3. Run the sample
