Microsoft Azure QuickStarts

This sample pack contains a collection of simple applications designed to help developers accomplish key scenarios with Microsoft Azure using the .NET SDK. Each sample covers a specific Azure technology, such as Blob Storage, Media Services or DocumentDB. The samples are intended to show the most common tasks relevant to each technology and do not attempt to comprehensively demonstrate everything that you may need. For complete details on each technology and the available features, please follow the links to the documentation pages that are included with each sample.

To run these samples, you'll need to have a Microsoft Azure subscription, Visual Studio, and the Azure SDK for .NET. If you don't have a Microsoft Azure subscription you can get a FREE trial account here.

Use the View samples tab to browse the available samples and download the source code as a .zip file. Once you have the Azure SDK installed, you can also access these QuickStart samples from the Visual Studio New Project dialog, under Templates \ Visual C# \ Cloud \ QuickStarts.

If you have any questions or feedback about the Azure QuickStarts, please use the Q and A tab and the Ratings capability included with each sample.

Sample index

Azure Active Directory Graph API
Demonstrates use of the AAD Grpah API to perform common read and write operations on Users, Groups, Group Membership, Roles, Tenant information, Service Principals, and Applications.
Azure Active Directory Web Authentication with OpenID Connect
This sample shows how to build a .Net MVC web application that uses OpenID Connect to sign-in users from a single Azure Active Directory tenant, using the ASP.Net OpenID Connect OWIN middleware.
Azure Media Services Transcoding
This template demonstrates how to use the Azure Media Services SDK to Transcode media assets. Media Services can be used to transcode media to HLS, DASH and Smooth Streaming formats.
Azure Service Bus Messaging With Queues
Simple demonstration of how to use Azure Service Bus Queues for messaging. This approach can be applied to de-couple backend + frontend roles and can also be used to load balance and load level processing for variable load scenarios
Azure Service Bus Messaging with Topics
Simple demonstration of how to use Azure Service Bus Topics to achieve PubSub pattern for distributing messages or events between producers and consumers
