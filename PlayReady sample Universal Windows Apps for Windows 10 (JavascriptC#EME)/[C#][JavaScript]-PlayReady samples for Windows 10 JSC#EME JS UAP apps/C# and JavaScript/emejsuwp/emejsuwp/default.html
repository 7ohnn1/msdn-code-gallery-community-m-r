This sample demonstrates:

  • Proactive License acquisition: Demonstrates the ability to pre-acquire licenses in EME scenarios within PlayReady.
  • LDL: Demonstrates playback with limited duration license (e.g. real time expiration and first play expiration = 60 seconds) followed by full license acquired 30 seconds after playback starts.
  • A/V sep: This sample uses separate content encryption keys for audio and video content.
  • DASH content: Demonstration of MPEG DASH Common Streaming Format playback.
