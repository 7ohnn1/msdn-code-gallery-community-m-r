Introduction

You can use PlayReady Content Protection to help you deliver audio and video content that is more secure and better protected from unauthorized redistribution.

Windows 10 now supports Enhanced Content Protection (ECP) with compatible Hardware. 

Prerequisites

  • Windows 10 RTM

  • Visual Studio 2015 RTM

  • Microsoft Universal Smooth Streaming Client SDK

    • Add Microsoft Universal Smooth Streaming Client SDK to the visual studio project’s reference in order to play smooth streaming (PIFF) content.  (This applies to the JS and XAML sample app only, the EME sample app doesn’t have PIFF content so it is not needed.)

 

Sample Details

The Solution 

This sample illustrates how to utilize PlayReady for playing protected content. 

 

Building the Sample

  1. Extract the solution from the Zip file

  2. Make sure you have installed the Microsoft Smooth Streaming Client SDK and Visual Studio 2015 RTM.

  3. Open the Solution File (.SLN) from that directory created in step #1.

  4. In Visual Studio, go into the Build menu, and select Build Solution.

  5. Ensure the build succeeds with zero errors.

  6. Hit F5, or go to the "Debug" Menu and select the "Start Debugging" Menu item to run. Make sure you have selected the correct architecture (x86, x64).

  7. This will run the default StartUp project. Change the StartUp project settings for the Project menu and repeat step#6 to debug another project.
