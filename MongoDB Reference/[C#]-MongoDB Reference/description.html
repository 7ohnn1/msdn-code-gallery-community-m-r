Introduction

Starting out with MongoDB, require a complete reference of C# MongoDB Driver with working code and demonstration, this application fits you. Select the topic you wants hands on in left pane and in right pane you can see the code, description, above all you can also run the sample code.

Building the Sample

Open the downloaded sample in Visual Studio, hit F5 to run the application. It will ask for a running MongoDB instance, if you have none leave them as such. But then any code snippet connecting with MongoDB is not going to work. If you start the MongoDB instance later with different settings you need to restart the application in order to change the values.

Description

It is a complete reference for MongoDB with C# and MongoDB using MongoConsole. You will get to know all the API calls available to you and doing the same thing using MongoConsole as well. We have started with basic like creating JSON(in console and BSONDocument in C#). We are also going to include MongoDB and C# driver calls. In broad sense this application is going to cover :

 

  • Creating and manipulating JSON (BSON) Documents.
  • Connecting with MongoDB and firing general commands
  • Creating (Inserting) documents in collection
  • Finding documents in collection
  • Deleting documents from collection
  • Updating documents in collection
  • Exporting data using mongodump, BSONDump, mongoexport
  • Importing data using mongorestore and mongoimport

 

Curently this application is in immature phase, please leave all your comments either here or on our GITHUB page. If you are interested in getting more tutorials added in this sample, then please rate it, else why should I put any effort.

Prerequisites

In order to open and build the project you should have Visual Studio 2010 or above.

MongoDB should be installed on machine for all the samples to work correctly or you will not be able to run samples which require MongoDB connectivity.
